Téléchargements du serveur: https://github.com/theolizard/remote-numpad-server/releases

Avec Remote Numpad, vous pouvez profiter du confort d'un pavé numérique sur votre ordinateur portable.

Vous pouvez utiliser ce pavé numérique de 3 façon différentes.

En utilisant le profil HID de Bluetooth (experimental), vous pouvez utiliser le Remote Numpad avec n'importe quel appareil equipé de Bluetooth (Android P ou plus requis).

En utilisant la connexion Bluetooth classique ou la connexion TCP (Wi-Fi), une application serveur doit etre allumée sur l'appareil. Le server peut etre telechargé depuis "https://github.com/theolizard/remote-numpad-server/releases" où vous trouverez également plus d'information.

IMPORTANT: Les connexions classiques Bluetooth et TCP (Wi-Fi) NE FONCTIONNERONT PAS si le serveur n'est pas allumé sur l'autre appareil.

Un didacticiel video se trouve à "https://youtu.be/xTfQparcgg4".

Le projet est open source et toutes contributions sont les bienvenues à "https://github.com/theolizard/remote-numpad".
